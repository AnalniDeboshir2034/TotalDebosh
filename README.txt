Рандомайзер — ЛР6
Файлы:
- index.html
- style.css
- script.js

Запуск:
Откройте index.html в браузере. Все функции работают локально. 
Сохранение некоторых данных происходит в LocalStorage (сохранённый список, история, исключения).

Описание функций:
- Числа: генерация случайных чисел, варианты с уникальностью.
- Список: выбор случайного элемента, перемешивание, сохранение.
- Цвет: генерация случайных цветов (HEX/RGB/HSL), клик по цвету — копирование.
- Кости: броски разных кубиков, advantage/disadvantage для d20.
- Имена: выбор с возможностью исключать выбранные до сброса.

Автор: Богоданный Финский и Богоподобный Вербицкийы
